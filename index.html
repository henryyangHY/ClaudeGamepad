Features

Everything you need to code hands-free

Native Swift + AppKit. No Electron. No Python. Just plug in and go.

🎮
Plug & Play Controllers
Auto-detects Xbox, PS5 DualSense, and any MFi-compatible controller via GCController. Switch between Xbox and PS5 label themes across the entire UI. Zero configuration required.
GCController API
🎙
Voice Input Pipeline
Wire any button to Voice Input. Auto-detects Chinese and English, transcribes locally with whisper.cpp, with optional LLM refinement via Ollama or any OpenAI-compatible endpoint.
whisper.cpp · Ollama
⚡
Quick Prompts
Hold LT or RT and press a face button to fire preset prompts instantly — run the tests, show me the diff, looks good, commit this and push. All 8 slots fully customizable.
8 configurable slots
🕹
Combo & Overlay Actions
Assign any button to a keyboard Combo (e.g. ⌘W, ⌘⇧]) to drive shortcuts and overlays directly, plus a floating HUD that shows live button feedback and transcription.
Fully remappable

Install

Up and running
in under 5 minutes

Build from source with Swift. No package manager drama. No Electron.

🍎 macOS 14.0 Sonoma+ 🎮 Xbox · PS5 · MFi Swift 5.9+ MIT License
# Clone and build
$ git clone https://github.com/henryyangHY/ClaudeGamepad.git
$ cd ClaudeGamepad
$ ./build-app.sh && cp -R ClaudeGamepad.app /Applications/
# Optional: higher-quality offline voice
$ brew install whisper-cpp
Grant Accessibility when prompted (for keyboard simulation) and Speech Recognition if using voice input. Connect your controller — the menu bar icon turns active.
